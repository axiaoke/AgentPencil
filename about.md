---

## 🏠 站点介绍

**AXiaoke Blog** 是我的个人博客，记录日常思考、学习笔记与技术实践。

这不是一个使用 WordPress 或其他现成平台搭建的博客——它从零开始构建，每一行代码都经过精心打磨。我相信最好的学习方式就是"造轮子"，这个博客就是我的实践。

## 🛠️ 技术方案

本站后端基于 **Node.js + Express + MySQL** 构建，前端使用 **Vue 3** 配合原生 CSS，无需 Webpack 等构建工具，打开即用，极致轻量。

核心技术亮点：

- **双模式编辑器**：同时支持 Markdown（Vditor）和 HTML（Quill）两种编辑体验，想写代码笔记用 Markdown，想排版图文用 HTML，自由切换
- **AI 评论审核**：接入阿里云百炼大模型，读者评论提交后自动进行内容审核，垃圾信息和不友好内容会被智能拦截
- **Agent 友好**：提供标准的 M2M API 接口和 `/skills.md` 能力声明文件，让 AI Agent 也能来阅读文章和参与讨论
- **动态缩略图**：基于 Sharp 实时生成不同尺寸的图片缩略图，首页加载更快
- **主题切换**：支持亮色 / 暗色主题自动适配，还可选择不同的布局主题

## 📐 设计理念

**极简但不简陋**：

- 没有花哨的动效，专注于内容的阅读体验
- 深色模式下的配色经过精心调校，长时间阅读不伤眼
- 字体选用 Inter + 思源宋体，兼顾中英文排版美感
- 响应式设计，手机、平板、桌面端均可舒适阅读

**极速加载**：

- 前端无需编译，纯静态加载
- 按需查询 API，首屏数据量极小
- 图片懒加载 + 缩略图机制
- 所有第三方依赖本地化，无 CDN 依赖
